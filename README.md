# Pandas Data Analysis Assignment

This repository contains my practice work and assignment solutions using *Python Pandas* for data manipulation and analysis.

The notebook demonstrates various fundamental operations used in real-world data analysis workflows.

## Topics Covered

The assignment includes practical exercises on:

- Creating DataFrames
- Data filtering and conditional selection
- Applying custom functions
- Temperature conversion using functions
- Creating new columns
- Boolean filtering
- Data slicing with head() and iloc
- GroupBy operations
- Aggregation functions (sum, count, min, max, mean)
- Working with date ranges in Pandas

## Key Concepts Practiced

### Data Filtering
Filtering rows based on conditions such as:

- Temperature ≤ 20
- Temperature ≥ 35
- Cities belonging to Union Territories

### Applying Functions
Using .apply() to:

- Check if a city is a Union Territory
- Convert temperature from Celsius to Fahrenheit

### GroupBy Analysis
Performing grouped analysis such as:

- Total salary per department
- Number of employees per department
- Minimum and maximum salaries

### Date Handling
Working with Pandas date ranges:

- Generating date sequences
- Checking length of date range
- Understanding DatetimeIndex
